# CSE6242 Final Project --Web

## Installation

* set google map API in map/templates/map.html    
$ cd map    
$ pipenv install --dev   
$ pipenv shell   
$ export FLASK_ENV=development   
$ export FLASK_APP=map
$ flask initdb   
$ flask run  
$ Running on(local) http://127.0.0.1:5000/
